Pointers for div

  1. width of a block level elemeny is 100% of the viewport width
  2. Height of a block isn't set by default
  3. Height and width can be explocitly set using CSS

Pointers for span

  1. width of an inline level elemeny is dependent on the content
  2. Height of an inline isn't set by default
  3. Height and width cannot be explocitly set using CSS
